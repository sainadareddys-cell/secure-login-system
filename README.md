# 🔐 Secure Login System

A simple and secure login system built using HTML, CSS, JavaScript, and Node.js.  
This project allows users to register and log in securely with password hashing.

---

## 🚀 Features

- User Registration
- User Login Authentication
- Password Hashing using bcrypt
- JSON-based database
- Simple and clean UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- bcrypt

---

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open terminal in project folder  
3. Run the following commands:

npm init -y  
npm install express bcrypt  
node server.js  

4. Open `index.html` in your browser  

---

## 🔐 Security

- Passwords are hashed using bcrypt  
- No plain text password storage  

---

## 📌 Notes

- Backend runs on: http://localhost:3000  
- Make sure Node.js is installed
